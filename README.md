# Employee Management System

A full-stack Employee Management System built with Next.js, React, and MySQL.  
This project was developed as a database course assignment.

---

## Features

- View all employees in a dynamic table
- Add new employee with validation
- Delete employee
- Update employee role
- Role-based extra fields (Frontend / Backend / Designer)
- GitHub profile validation
- Modal-based employee creation form
- Full CRUD operations using API routes

---

## Tech Stack

### Frontend

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

### Backend

- Next.js API Routes
- MySQL
- SQL JOIN Queries
